# web-fanhouse-archive

本仓库用于归档和发布多个独立 HTML 页面，内容涵盖日常记录、工具页面、信息展示等。所有页面均为静态 HTML 文件，不依赖后端服务，可直接在浏览器中打开查看。

## 项目简介

`web-fanhouse-archive` 是一个个人归档项目，旨在收集并发布一些独立的 HTML 页面。这些页面可能包含个人整理的信息、简单的工具页面或特定主题的展示页。项目本身不针对任何特定网站或域名，所有页面均为独立存在。

## 目录结构

```
web-fanhouse-archive/
├── index.html          # 仓库主页，列出所有归档页面链接
├── pages/              # 存放各独立 HTML 页面
│   ├── page1.html
│   ├── page2.html
│   └── ...
├── assets/             # 共用资源文件（图片、样式、脚本等）
│   ├── css/
│   ├── js/
│   └── images/
└── README.md           # 本文件
```

- `index.html`：仓库入口页面，提供所有归档页面的导航。
- `pages/`：存放各个独立 HTML 页面，每个页面为一个完整归档单元。
- `assets/`：存放页面共用的资源文件，如样式表、脚本和图片。

## 页面归档说明

每个归档页面均为独立的 HTML 文件，内容结构完整，不依赖外部服务。页面内容可能随时间更新或补充，但不会频繁修改。归档页面主要用于个人记录和分享，不保证内容的时效性或准确性。

## 维护说明

- 本仓库不定期更新，新增页面或对现有页面进行修订。
- 新增页面时，会同步更新 `index.html` 中的导航链接。
- 若发现页面内容有误或链接失效，欢迎通过 Issue 提出，但无法保证及时修复。
- 本仓库不接受外部直接提交内容，仅由仓库维护者管理。

## 使用方式

1. 克隆或下载本仓库到本地。
2. 使用任意浏览器打开 `index.html` 或 `pages/` 目录下的任意 HTML 文件即可查看。
3. 也可通过 GitHub Pages 等服务部署后在线访问（需自行配置）。
